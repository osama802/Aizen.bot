```bash

3. افتح F-Droid وابحث عن "Termux"
```

```bash
رابط التحميل المباشر:
```

## 🔧 خطوات التثبيت (خطوة بخطوة)

### الخطوة 1️⃣: فتح Termux

```
افتح تطبيق Termux على هاتفك
ستظهر واجهة سوداء - هذا طبيعي ✓
```

### الخطوة 2️⃣: تحديث النظام

انسخ الأمر التالي والصقه في Termux:

```bash
apt update && apt upgrade -y
```

اضغط `Enter` وانتظر حتى ينتهي (قد يستغرق عدة دقائق)

### الخطوة 3️⃣: تثبيت Node.js و NPM

```bash
apt install -y nodejs npm
```

تحقق من التثبيت:

```bash
node --version
npm --version
```

يجب أن تظهر إصدارات للاثنين ✓

### الخطوة 4️⃣: تثبيت Git

```bash
apt install -y git
```

### الخطوة 5️⃣: استنساخ المشروع

```bash
git clone https://github.com/osama802/Aizen.bot.git
```

ثم ادخل مجلد المشروع:

```bash
cd Aizen.bot
```

### الخطوة 6️⃣: تثبيت المكتبات

```bash
npm install
```

**⏳ سيستغرق هذا عدة دقائق، انتظر حتى ينتهي تماماً**

## 🚀 التشغيل

### تشغيل بسيط

```bash
node index.js
```

### تشغيل في الخلفية (nohup)

```bash
nohup node index.js > bot.log 2>&1 &
```

عرض السجل:
```bash
tail -f bot.log
```

### تشغيل متقدم مع tmux

تثبيت tmux:
```bash
apt install -y tmux
```

إنشاء جلسة:
```bash
tmux new-session -d -s aizen "cd ~/Aizen.bot && node index.js"
```

عرض الجلسة:
```bash
tmux attach-session -t aizen
```

الخروج (اضغط `Ctrl+B` ثم `D`):
```
[جلسة البوت تستمر في الخلفية]
```

### تشغيل مع screen

تثبيت screen:
```bash
apt install -y screen
```

إنشاء جلسة:
```bash
screen -S aizen -d -m bash -c "cd ~/Aizen.bot && node index.js"
```

عرض الجلسة:
```bash
screen -r aizen
```

الخروج (اضغط `Ctrl+A` ثم `D`):
```
[جلسة البوت تستمر في الخلفية]
```

## ⚠️ حل المشاكل الشائعة

### المشكلة 1: "command not found: node"

**الحل:**
```bash
apt install -y nodejs npm
```

### المشكلة 2: "command not found: git"

**الحل:**
```bash
apt install -y git
```

### المشكلة 3: فشل مسح QR Code

**الحل:**
```bash
# حذف بيانات المصادقة القديمة
rm -rf .wwebjs_auth
rm -rf .wwebjs_cache

# إعادة التشغيل
node index.js
```

### المشكلة 4: "EACCES: permission denied"

**الحل:**
```bash
chmod -R 755 .
npm install --unsafe-perm
```

### المشكلة 5: البوت لا يستقبل الرسائل

**تحقق من:**
1. الاتصال بالإنترنت
2. تسجيل الدخول صحيح في واتساب
3. إعادة تشغيل البوت

```bash
# إعادة التشغيل
node index.js
```

### المشكلة 6: "Timeout" عند مسح QR

**الحل:**
```bash
# زيادة مهلة الاتصال - اضغط Ctrl+C لإيقاف البوت
# عدّل في index.js السطر 2 من "client.initialize"

node index.js
```

### المشكلة 7: استهلاك ذاكرة عالي

**الحل:**
```bash
# تحديث المكتبات
npm update

# أعد التثبيت
rm -rf node_modules package-lock.json
npm install
```
